# 3D Printed Enclosure

Put 3D-print enclosure files here.

Recommended structure:
- `stl/` for printable meshes
- `step/` for editable CAD exchange files
- `sources/` for native CAD project files (Fusion 360, FreeCAD, etc.)

Use clear file names with revision suffixes, for example:
- `servo_tester_case_top_revA.stl`
- `servo_tester_case_bottom_revA.stl`
